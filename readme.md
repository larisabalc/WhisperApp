## Setup Steps

### 1. Install requirements

Install Streamlit:

```bash
pip install -r requirements.txt
```

### 2. Install Whisper

Install OpenAI Whisper:

```bash
pip install openai-whisper
```

### 3. Install FFmpeg

Whisper uses FFmpeg to process media. Install according to your system:

Windows (winget):

```bash
winget install FFmpeg
```
